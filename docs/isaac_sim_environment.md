# Isaac Sim Environment Setup

This document provides an introduction to NVIDIA Isaac Sim's interface and Omnigraph system. By working through these instructions, you'll become familiar with the core components needed to effectively use the `haptic_teleop_core` ROS2 package for haptic teleoperation tasks.

## Overview

Isaac Sim is a robotics simulation platform that uses a visual node-based system called Omnigraph for creating complex behaviors and workflows. This guide covers:

- Navigating the Isaac Sim interface
- Understanding Omnigraph for visual programming
- Setting up your environment for ROS2 integration
- Preparing to work with haptic teleoperation systems

By the end, you'll be ready to deploy and interact with the `haptic_teleop_core` package.
